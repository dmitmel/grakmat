linesWithSeparators

fun String.linesWithSeparators(): List<String>

Splits string to list of lines, which also includes used separators.

Examples:

a\n
b\n   ====> ["a\n", "b\n", "c"]
c
a\r
b\r   ====> ["a\r", "b\r", "c\r"]
c\r
a\r\n
\r\n  ====> ["a\r\n", "\r\n", "b\r\n", "c"]
b\r\n
c
